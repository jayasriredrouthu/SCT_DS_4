## Task 04: Traffic Accident Data Analysis

### Project Overview

This project was completed as part of the **SkillCraft Technology Data Science Internship – Task 04**.

The objective of this project is to analyze **Traffic Accident Data** to identify patterns related to **road conditions**, **weather**, and **time of day**, while visualizing accident hotspots and contributing factors using Python.

---

## Task Objective

* Perform data cleaning and preprocessing.
* Analyze accident patterns based on weather and road conditions.
* Explore accident trends across different times of the day.
* Identify accident hotspots using data visualization.
* Generate meaningful insights through Exploratory Data Analysis (EDA).
